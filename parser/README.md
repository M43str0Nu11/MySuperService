# Парсер погоды для Новосибирска

## 3 источника данных:

### 1. Open-Meteo
- ✅ Бесплатный, без регистрации
- API: https://api.open-meteo.com
- Координаты: 55.0084, 82.9357

### 2. wttr.in
- ✅ Бесплатный, без регистрации
- API: https://wttr.in
- Почасовой прогноз JSON

### 3. 7Timer!
- ✅ Бесплатный, без регистрации
- API: http://www.7timer.info
- Простой JSON API

## Как работает:

- Каждые 30 минут парсит погоду
- Определяет период (утро/день/вечер)
- Записывает в БД
- БД автоматически усредняет через VIEW

## Запуск:

```bash
docker-compose up -d --build
```

## Логи:

```bash
docker logs -f mysuperservice-parser
```
